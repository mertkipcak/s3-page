How This Page is Deployed

This webpage is deployed using AWS S3 and GitHub Actions. Here is how the process works:

  • Code Repository: The HTML, CSS, and JavaScript files are stored in a GitHub repository.
  • Automation with GitHub Actions: Whenever changes are pushed to the repository, GitHub Actions automatically deploys the updates to an S3 bucket.
  • AWS S3 Hosting: The S3 bucket is configured for static website hosting, which serves the files as a website.
  • Custom Domain: The domain page.mertkipcak.com is pointed to the S3 bucket to make the page accessible via a custom URL.
